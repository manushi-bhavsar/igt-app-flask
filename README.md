# IOWA Gambling Test

This is a project done for my capstone project at my university. It is a simple app that runs IGT in order to place the user on a spectrum.

## Installation
Ensure you have Python installed on your system. 

Install Flask:
Use the package manager [pip](https://pip.pypa.io/en/stable/) to install flask.

```bash
pip install flask
```
Navigate to the Project Directory using cd in your command prompt

Using the flask command:
```bash
flask run
(or)
python -m flask run
```
Open your web browser and navigate to the address provided in the terminal
